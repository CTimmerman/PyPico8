Run [PICO-8](https://www.lexaloffle.com/pico-8.php) demos in Python.

## Install on Windows

```cmd
python -m venv .venv
.venv\Scripts\activate
python -m pip install -r requirements.txt
```

The src folder contains demos ported from Twitter/X etc.

## Use

```cmd
.venv\Scripts\activate
python src\snek.py
```
